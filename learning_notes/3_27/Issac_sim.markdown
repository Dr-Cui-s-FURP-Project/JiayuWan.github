# Week report -- 3/27

## Issac Sim
1. 创建对象
    - Create -> Shapes -> Cubes  
      
2. 调整对象：
    - E：转换为旋转模式，点击轴线按住旋转。多次点击可在全局状态和物体本地状态下切换旋转。本地状态下，图标显示为橙色
    - R：转换为缩放模式，按住轴线进行缩放
    - W：转换为移动模式，按住物体进行移动。再按一次，在物体的轴线上进行移动。多次点击可在全局状态和物体本地状态下切换平移
    - ESC：取消选中的物体

3. 属性窗口：
    - Translate：可调整X，Y，Z轴来平移物体。双击值可平移到指定位置。点击值后蓝色方块可还原为(0,0,0)
    - Orient：调整相应值来调整物体旋转角度
